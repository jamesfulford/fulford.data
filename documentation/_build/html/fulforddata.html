add(unpackers, *args)[source]¶
Adds valid entries (passed later) in format given by unpackers
to self.entries.

Invalid entries added to self.rejects Un-unpackable entries added to self.packed

Passes extra args to each getter.

get_dictionaries(form, *args)[source]¶

Given dictionary (form), runs accessor values on each of the entries stored.

Example form is {

“time”: access(“x”), “measures”: {

“pirates”: access(“y1”, default=0), “CO 2”: access(“y2”, mapping=lambda n: round(n, 3))

}

}

get_rows(columns, fill='', *args)[source]¶

Returns data as rows according to given columns

Examples given use columns: [

(0, access(“x”, default=0, mapping=IDENTITY)), (2, access(“y”, mapping=abs))

]

(Columns can also be a dictionary)

get_writeable_rows(columns, fill='', delimiter=', ', subdelimiter='|', headers=True, *args)[source]¶

Returns list of delimited strings from self.get_rows()

Lists of lists will be flattened. (unable to represent lists of lists in strings)

validate(template, **contexts)[source]¶

Using each template, which has a similar structure to each entry, will apply each leaf function on each entry in self.entries.
